
Customer Management System – Requirements

1. System should be able to maintain customer data which includes name, age, geography and products bought.

2. System should record for data for all types of Customer:

	=> Paid Customer who have bought products.
	=> Enquiry Customer who have not bought products but done enquiry.

3. System can have different discount calculation as per customer:
							
	=> Normal customer will not have any discounts.
	=> If your age is above 60, you get 10% senior discount.
	=> If you buy on weekends, you get 2% discount.
	=> Customers from Mumbai and Pune will get 1% discount.

4. There are different delivery mechanisms of products to customer:

	=> Home delivery through Courier.
	=> Pickup from shop.
